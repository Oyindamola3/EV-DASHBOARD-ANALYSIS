🚗 Electric Vehicle Market Analysis Dashboard (Excel)
📌 Project Overview
This project explores Electric Vehicle (EV) market trends through data cleaning, analysis, and an interactive dashboard developed in Microsoft Excel. The objective was to transform raw EV data into meaningful insights that support understanding of EV adoption patterns, market growth, and key industry trends.
🎯 Objectives
* Analyze EV market trends and adoption patterns.
* Identify key factors influencing EV growth.
* Build an interactive dashboard for easy data exploration.
* Present insights using clear and interactive visualizations.
🛠️ Tools & Techniques Used
Microsoft Excel
* Data Cleaning
* Data Transformation
* Descriptive Statistics
* Pivot Tables
* Pivot Charts
* Slicers
* Interactive Dashboard Design
📂 Dataset
The dataset contains Electric Vehicle-related information used to analyze market trends, vehicle categories, and other key factors influencing EV adoption.
🔍 Data Cleaning Process
The following data preparation steps were carried out:
* Removed duplicate records.
* Checked and handled missing values.
* Corrected inconsistent data formats.
* Standardized data for accurate analysis.
* Prepared the dataset for visualization.
📊 Dashboard Features
The dashboard includes interactive visualizations that allow users to:
* Explore EV trends using slicers.
* Compare different categories and key metrics.
* Identify patterns and relationships within the dataset.
💡 Key Insights
* EV adoption has shown significant growth over time.
* Certain EV categories and segments dominate the market.
* Interactive filters make it easier to explore trends across different variables.
